# RCCG S&P Parish — Official Website

> **"A place of worship, discipleship, and community in the heart of Lagos."**

The official website of the **Redeemed Christian Church of God — Salvation & Praise Parish**, located at 59 Peter Agha, Oke-Afa, Isolo, Lagos, Nigeria.

This project was conceived, designed, and built entirely by members of the **Teen Church** of RCCG S&P Parish — a testament to the talent, dedication, and faith of the next generation of builders in the house of God.

## 🌐 Project Overview

This is a fully responsive, multi-page static website built with plain **HTML5**, **CSS3**, and **Vanilla JavaScript** — no frameworks, no build tools, just clean, handcrafted code.

The site serves as the digital home of the parish, providing members and visitors with:

- Information about the church, its history, mission, and leadership
- Sermon archives with video and audio playback
- Event listings and a live service countdown
- A photo gallery of church life and ministry moments
- Ministry and department profiles
- Online giving with bank transfer details
- Prayer request submission
- Volunteer/ministry sign-up
- Blog and devotional content
- Contact information and an embedded map

## ✨ Features

- **Fully Responsive** — optimised for mobile, tablet, and desktop
- **Mobile Navigation** — slide-in hamburger menu with animated dropdowns
- **Live Countdown** — counts down to the next scheduled service in real time
- **Sermon Player** — embedded YouTube videos + fixed audio player bar with keyboard controls
- **Gallery** — filterable grid (All / Sunday Service / Outreach / Events / Ministries) with a full-screen lightbox, swipe support, and keyboard navigation
- **Blog Modal** — articles open in an in-page modal with share buttons (WhatsApp, Facebook, copy link)
- **Online Giving** — giving type tabs, preset amounts, bank account details with copy-to-clipboard
- **Forms** — prayer request, volunteer sign-up, and contact forms with client-side validation
- **Scroll Reveal Animations** — elements animate into view as the user scrolls
- **Service Moments Slideshow** — auto-playing Ken Burns slideshow on the homepage
- **Team Lightbox** — click any team card to view the full photo
- **WhatsApp Float Button** — quick access to chat with the church
- **Accessibility** — semantic HTML, ARIA labels, keyboard navigation, focus management throughout

## 🚀 Running Locally

This is a static website — no server or build step required.

**Option 1 — Open directly:**
```
Double-click index.html in your file explorer
```

**Option 2 — Use the included batch file (Windows):**
```
Double-click start_server.bat
```

**Option 3 — VS Code Live Server:**
1. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
2. Right-click `index.html` → **Open with Live Server**
